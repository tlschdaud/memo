입출력(I/O)

1. 자바에서의 입출력
  1) 입출력이란?
     Input(입력), Output(출력)

  2) 스트림(stream)
    데이터 통로
    입력 스트림(Input - 키보드, 네트우크)
    출력 스트림(Output - 화면, 프린트, 파일) 

2. 바이트기반 스트립
  1바이트 기반 스트림(데이터를 1바이트씩 주고 받는)
  stream으로 끝나는 클래스 -> 바이트 단위

1) 입력 스트림
  InputStream
  - 기반 스트림 
	-직접 데이터에 접근하는 스트림
	ByteArrayInputStream : 메모리
	FileInputStream : 파일
	  - int read()
	     - read() 호출 할 때마다 1바이트씩 이동하면서
	     - 끝까지 다 읽게되면 -1 반환
	    0 ~ 255
	IOException
	FileNotFoundException


  - 보조 스트림
	- FilterInputStream
	- 직접 데이터 접근 X
	- 생성자 매개변수 : InputStream
	   - 기반 스트림 + 보조기능(추가기능)
	- BufferedInputStream
	   - 8k

2) 출력 스트림
  OutputStream
  - 기반 스트림
	-직접 데이터에 접근하는 스트림
	ByteArrayOutputStream : 메모리
	FileOutputStream : 파일

  - 보조 스트림
	- 직접 데이터 접근 X
	- 생성자 매개변수 : OutputStream
	   - 기반 스트림 + 보조기능(추가기능)
	- DataOutputStream // 기본 자료형

3. 문자기반 스트림(문자단위 - 2byte, 3byte)

  1) 입력 스트림
      Reader
          - read(char[] cbuf, int off, int len)

      - 기반 스트림
         - 직접 데이터 접근(문자 단위) 읽기
         - CharArrayInputStream : 메모리
         - FileReager : 파일
         - StringReader : 메모리

      - 보조 스트림
         - 직접 데이터 접근 X
         - 기반 그트림 + 추가 기능(문자 단위 처리)
         - BufferedReader : 버퍼(8Kb)

  2) 출력 스트림
       Writer
          -write(char[] cbuf, int off, int len)

      - 기반 스트림
         - 직접 데이터 접근(문자 단위) 출력
         - CharArrayOutputStream : 메모리
         - FileWriter : 파일
         - StringWriter : 메모리

      - 보조 스트림
         - 직접 데이터 접근 X
         - 기반 스트림 + 추가 기능(문자 단위 처리)
         - BufferedWriter : 버퍼(8kb)
	 문장단위로 읽고 처리하는 기능

5.PrintStream / PrintWriter / 버퍼 내장
      - println() -> 줄개행 문자(\r\n)
      - println -> 기본자료형 -> 기본자료형 + String 출력 + 줄개행 문자(\r\n)
      - printf -> 형식화 해서 출력
      - print -> 기본 자료형 + String 출력

메타문자 : 특정 동작을 발생시키는 문자
\r : carrige return : 커서를 가장 처음 위치로 이동
\n : new line : 줄 개행
